L'Équipe

19 agents spécialisés — de la stratégie au déploiement. Les tags "Requis" indiquent les champs de project-context.md nécessaires à chaque agent.

Installation

3 étapes, 2 minutes — l'équipe s'adapte à votre projet

Nouveau Nouveau projet

Pas encore de code — vous partez de zéro

1 Créez un repo git et ouvrez Claude Code dessus
2 Copiez ce prompt :
"Clone https://github.com/thomasissa-png/Agent-Team dans /tmp et installe l'équipe Gradient Agents dans ce projet. Nouveau projet."
3 Remplissez project-context.md puis lancez @orchestrator
Résultat :
19 agents dans .claude/agents/ + CLAUDE.md + project-context.md (template) + docs/ et src/

Existant Projet existant

Du code est déjà en place — rien n'est écrasé

1 Ouvrez Claude Code à la racine de votre repo git
2 Copiez ce prompt :
"Clone https://github.com/thomasissa-png/Agent-Team dans /tmp et installe l'équipe Gradient Agents. Projet existant, ne rien écraser."
3 Remplissez project-context.md puis lancez l'agent dont vous avez besoin
Résultat :
19 agents ajoutés + CLAUDE.md fusionné + project-context.md (template) — src/, docs/, package.json intacts

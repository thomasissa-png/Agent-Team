Installation

Deux scénarios selon votre situation — l'équipe s'adapte à votre projet

Nouveau Nouveau projet

Pas encore de code — vous partez de zéro

Ouvrez Claude Code sur le dossier de votre nouveau projet (qui doit etre un repo git), puis :

"Clone https://github.com/thomasissa-png/Agent-Team dans /tmp et installe l'équipe Gradient Agents dans ce projet. C'est un nouveau projet. Installe .claude/agents/ à la racine du repo git."
  • 19 agents copiés dans .claude/agents/ à la racine du repo git
  • CLAUDE.md copié à la racine du repo git
  • project-context.md créé (template vierge)
  • docs/ et src/ créés si absents
  • Premier réflexe : @orchestrator pour planifier tout le projet

Existant Projet existant

Du code est déjà en place — rien n'est écrasé

Ouvrez Claude Code à la racine de votre repo git (pas dans un sous-dossier), puis :

"Clone https://github.com/thomasissa-png/Agent-Team dans /tmp et installe l'équipe Gradient Agents dans ce projet. C'est un projet existant, ne rien écraser. Installe .claude/agents/ à la racine du repo git, pas dans un sous-dossier."
  • 19 agents ajoutés dans .claude/agents/ à la racine du repo git
  • CLAUDE.md fusionné avec l'existant
  • project-context.md créé (documenter l'existant)
  • src/, docs/, package.json pas touchés
  • Premier réflexe : l'agent spécifique au besoin (@fullstack, @seo, @qa…)
